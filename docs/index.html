UIMan

Fast and flexible solution for development and UI management with MVVM pattern
download .ZIPdownload .TGZ

UIMan is fast and flexible solution for UI development and management with MVVM pattern. UIMan is designed most suitable for mobile game.

Features

  • Support Databinding, Observable and MVVM for implementing UI with uGUI.
  • Prefab base UI, async loading.
  • Auto Code Generation.
  • UI flow and layer management system.
  • UI Events.
  • Customizable loading indicator.
  • Easy to extend and customize.
